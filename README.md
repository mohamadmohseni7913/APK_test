# سیستم تیکتینگ سازمانی (فرانت‌اند)

یک سیستم مدیریت تیکت ساده و قابل توسعه با **React + TypeScript + Tailwind CSS** و مدیریت داده با **TanStack Query (React Query)**.

## مقدمه

این پروژه بخش فرانت‌اند یک سیستم تیکتینگ سازمانی است که با هدف پیاده‌سازی یک رابط کاربری تمیز، سریع و کاربرپسند توسعه داده شده. چون در زمان توسعه به API واقعی (DRF) دسترسی نداشتم، از **json-server** برای شبیه‌سازی بک‌اند و دیتابیس استفاده کردم.

**ویژگی‌های اصلی:**
- ثبت‌نام و ورود کاربر (mock با json-server)
- ایجاد تیکت توسط کاربر عادی
- مشاهده لیست تیکت‌ها (کاربر فقط تیکت‌های خودش، ادمین همه تیکت‌ها)
- فیلتر بر اساس وضعیت، اولویت و جستجو
- تغییر وضعیت تیکت توسط ادمین
- بروزرسانی خودکار لیست بدون نیاز به رفرش صفحه (با React Query)
- طراحی ریسپانسیو و مدرن با Tailwind CSS

## تکنولوژی‌های استفاده‌شده

- **React 19** + **TypeScript** – برای ساختار قوی و جلوگیری از خطاهای رایج
- **Tailwind CSS** – استایل‌دهی سریع و انعطاف‌پذیر
- **TanStack Query (React Query)** – مدیریت درخواست‌ها، کشینگ، invalidate و بروزرسانی خودکار
- **React Hook Form** – مدیریت فرم‌ها و اعتبارسنجی سمت کلاینت
- **React Router v6** – روتینگ و ناوبری
- **react-hot-toast** – نمایش نوتیفیکیشن‌های کاربرپسند
- **json-server** – شبیه‌سازی بک‌اند و دیتابیس (برای توسعه و تست)
- **js-cookie** – مدیریت توکن در کوکی (امنیت بیشتر نسبت به localStorage)

## دلیل انتخاب معماری و ابزارها

- **React Query**: بهترین انتخاب برای مدیریت داده‌های سمت سرور بود. کشینگ خودکار، مدیریت loading/error/success، invalidateQueries برای بروزرسانی لیست بعد از ایجاد/ویرایش، و امکان refetchInterval برای بروزرسانی دوره‌ای (در آینده می‌تونه realtime بشه).
- **Tailwind CSS**: سرعت توسعه UI رو خیلی بالا برد و کد استایل تمیز و قابل نگهداری نگه داشت.
- **TypeScript**: از خطاهای رایج در زمان توسعه جلوگیری کرد و کد رو خواناتر کرد.
- **React Hook Form**: فرم‌ها رو خیلی راحت و با اعتبارسنجی قوی مدیریت کرد.
- **json-server**: چون بک‌اند واقعی (DRF) هنوز آماده نبود، بهترین راه برای شبیه‌سازی API RESTful و تست فرانت بود.

## نحوه اجرا (محلی)

1. کلون پروژه:
   ```bash
   git clone <repository-url>
   cd <project-folder>

2. نصب وابستگی ها :
  ```bash
  npm install

3. ران کردن json-server (شبیه‌سازی بک‌اند):
  ```bash
  npm run server

4. ران کردن فرانت‌اند:
   ```bash
   npm run dev
